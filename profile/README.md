## What Are We Building?

[**Instill AI**](https://www.instill-ai.com) is a multi-agent framework, that allows all kinds of unstructured data to be processed and transformed into structured data to help knowledge exploration, discovery and analysis, based on the open-source project [**Instill Core**](https://github.com/instill-ai/instill-core), a full-stack AI platform for data, model and pipeline orchestration, designed to streamline every aspect of building versatile AI task agents.
